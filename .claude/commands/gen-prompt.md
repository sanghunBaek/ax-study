# /gen-prompt

현재 DB의 최신 통계를 기반으로 LLM 추천 프롬프트를 미리보기합니다.

## 목적
`POST /api/recommend/llm` 에서 Claude에게 실제로 전달되는 프롬프트가
어떤 통계 컨텍스트를 포함하는지 개발 중에 확인하는 용도.

## 실행
1. `backend/src/main/resources/prompts/` 의 프롬프트 템플릿 파일 읽기
2. 최근 20회차 통계 데이터로 템플릿 채우기
3. 완성된 프롬프트 전체 출력

프롬프트가 너무 길거나 불필요한 데이터가 포함됐는지 검토하는 데 사용.
