# /analyze

최근 N회차 로또 데이터 통계를 분석하고 요약합니다.

## 사용법
```
/analyze [회차수=20]
```

## 출력 예시
- Hot 번호 Top 10 (최근 20회차 기준)
- Cold 번호 Top 10 (가장 오래 미출현)
- 번호대별 분포 (1-10, 11-20, 21-30, 31-40, 41-45)
- 홀짝 비율
- 최근 3회 연속 출현 번호

## 구현
`GET /api/stats?range=$ARGUMENTS` 를 호출해서 결과를 분석하고 요약해줘.
결과는 마크다운 테이블로 보기 좋게 출력해줘.
